# 工具首页 需求设计

## 1. 概述 (Overview)

工具首页用于**路由**到各个具体功能模块

## 2. UI设计

**界面居中展示** 从上向下，依次展示内容：

1. `AI助手搜索`。包含左侧搜索框和右侧搜索按钮  --> “当点击搜索按钮”后路由到AI助手功能
2. `常用AI助手命令`。以绿色标签展示，每行展示3个  --> 点击后路由到AI助手功能
3. `常用工具`。网格方式展示，类似于windows11开始菜单，每个网格内为图标和对应名称 --> 点击后路由到对应的工具

当前，`常用工具` 包括：

1. markdown文本编辑
2. 图片查看

**注意**，`常用工具`的图标使用element-plus的图标
